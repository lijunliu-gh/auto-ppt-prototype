# Release Draft: v0.2.0

## Title

`v0.2.0 - Open-source release preparation and agent-ready PPT workflow`

## Suggested Release Notes

This release turns Auto PPT Prototype into a publishable open-source repository for early adopters and integrators.

### Highlights

- prompt-to-deck PowerPoint generation
- revise flow for existing decks
- source-aware planning from local files and URLs
- structured slide source metadata with presenter-note source export
- JSON skill workflow for agent integration
- local HTTP service wrapper
- English, Simplified Chinese, and Japanese documentation

### Repository Readiness Improvements

- added MIT license
- added contribution and security documentation
- added issue templates and pull request template
- added smoke CI workflow
- aligned repository and skill naming to `auto-ppt-prototype`

### Notes

- this is still an experimental prototype
- the repository is an agent backend, not a full research agent
- image OCR, stronger provenance mapping, and production-grade template control are still future work
